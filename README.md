<h1>스마트 얼굴인식 도어락_[졸업작품]</h1>


<br/><br/><br/>

<h2>기획의도</h2>
뉴스나 신문을 보면 다양한 범죄에 관한 내용을 접할 수 있을것이다. 어떻게 하면 나 자신 , 가족 , 친구 등 모두의 안전을 확실하 보장받을 수 있을까 생각을 하다가 "스마트 얼굴인식 도어락"이라는 주제를 선정하게 되었다. 라즈베리파이를 통한 openCV를 사용하여 사용자의 얼굴을 등록하고 딥러닝 , 머신러닝을 활용하여 등록된 사용자의 얼굴정보와 일치할 경우 도어락이 열리는 스마트 도어락을 설계하였고, 실시간으로 카메라 화면을 볼 수 있어 cctv로도 활용할 수 있도록 기술을 개발하였다.
